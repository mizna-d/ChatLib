ChatLib Documentation.

Setting Up

ChatLib is a very easy to use front-end library and only requires a few dependencies before we can get started. ChatLib uses jQuery so it is necessary for you to include jQuery as a script in the head of your HTML like so: <script defer src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script> ChatLib also requires certain fonts which must also be imported as well. The code below should be added to the head of your HTML to import this dependency: <link rel="preconnect" href="https://fonts.googleapis.com">

<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>

<link href="https://fonts.googleapis.com/css2?family=Cedarville+Cursive&family=Comforter+Brush&family=Cookie&family=Satisfy&family=Varela+Round&display=swap" rel="stylesheet"> Lastly, we must import the ChatLib Library itself. You can download the code here. Importing the library can be done by including this following script in the head of your HTML like so: <script defer type="text/javascript" src="ChatLib.js"></script> Now we have finished setting up and we start creating chat interfaces right away! An example of a basic implentation can be found on the home page

Global Variables

The global variables available for use

variable name default type description
userMessages [] Array Stores all messages sent by the user which allows the developer to have access to all previous user input
themeSets [defaultTheme, businessTheme, coffeeTheme, customTheme] Array Stores all themeSets available for the developer to set the current theme as.

Private Variables

Though these are not available for use, it is important to know the default values they are set as. They can be changed through methods found in the next section.

variable name default type description
botIcon '' String Stores the filePath to the bot's profile picture
userIcon '' String Stores the filePath to the user's profile picture
minimizedOnLoad false Boolean Setting for the chat window to be minimized when the page loads
botDelay 1.5 Number Delay time in seconds before each message sent by the bot
userDelay 0.5 Number Delay time in seconds before each user input is presented in the user window
currentTheme defaultTheme Theme Object The theme the chat window should present

Methods

The methods available for developers to use to create quick and easy chat interfaces.

method parameters types description
createChatWindow() none none Initializes the window where the chatbot will reside.
addBotMessage() text: The text within the bot's message
nextFunction: The function that should follow after 		String, function Adds bot messages into the chat window.
addUserMessage() text: The text within the user's message
nextFunction: The function that should follow after		 String, function Adds user messages into the chat window.

String Changes the theme of the chat window to best fit any site. Currently there are 3 themes available "defaultTheme", "coffeeTheme" and "businessTheme". Must be set before the chatWindow is initialized.
setMinimizeOnLoad() isMinimizedOnLoad: Enables minimization on page load
Boolean Sets the chat window to either be minimized when page is loaded or expanded. Must be set before the chatWindow has been initialized.

messageStyles:Holds changes to the message styles, if no changes, pass null
chatHeaderStyles:Holds changes to the chatHeader styles, if no changes, pass null
questionStyles: Holds changes to the questions styles, if no changes, pass null
 Theme Objects Allows to create a custom theme for chat window. Must be set before the chatWindow has been initialized. More details on the format of each object and customization can be found at the bottom of this page.
setBotIcon() botIcon: String representing filepath to bot's profile piciture
String Sets the bot’s profile picture shown in the chat. Must be set before any message from the bot is sent.
setUserIcon() userIcon: String representing filepath to user's profile piciture
String Sets the user’s profile picture shown in the chat. Must be set before any message from the user is sent.

Number Sets the delay time between messages sent by the bot.
addTextQuestion() question: Question presented as a bot message before text input is prompted
placeholder: Placeholder text for input field
nextFunction: The function that should follow after
String, String, function Makes text input field to visble and prompts the user to type their answer to a question. When submitted, user input will be add to the chat window in the form of a chat bubble.
addDragDropQuestion() question: Question presented as a bot message before drag adn drop is prompted
choices: Categories the inputs can be dragged into
items: Inputs the user can drag into categories
nextFunction: The function that should follow after

nextFunction: The function that should follow after

Setting Up a Custom Theme

ChatLib allows for full customizability so you chatUI will always seamlessly integrate into the rest of your website. Below is the full configuration of the defaultTheme. When setting up your own custom theme, the custom theme has already been preset as the default theme, so when customizing you would be making edits to this set of styles, rather than starting from scratch.

In order to create your own custom theme, you must create each of the objects found within the Theme Object and edit the properties to you liking. The setCustomTheme() function allows you to make edits to just one, or all of the objects found within a Theme Object for convenience.
